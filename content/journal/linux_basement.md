# 文件管理命令

**命令区分大小写**

## 1.查看文件列表
ls *查看当前目录下的文件列表*


ls / *查看指定目录下的文件*


ls -l *查看详细信息*

## 2.切换目录
cd 目标文件夹 

## 3.查看当前命令所在的目录
pwd
 ### 特殊目录符号
 ～ *当前用户的home目录*


 . *当前目录*


 .. *上一级目录*

## 4.新建文件夹及文件
mkdir 文件夹名 *在当前位置新建文件夹*


mkdir -p /a/b/文件夹名 *在指定目录位置，创建文件夹，并创建父文件夹*


touch 文件名 *在当前目录下新建文件*

## 5.删除文件
rm 文件名 *删除文件*


rm -r 文件夹 *删除文件夹*


rm -rf 文件名 *强制删除不询问*

## 6.拷贝文件
cp 原文件 新文件 *拷贝文件*


cp -r 源文件夹 新文件夹 *拷贝文件夹*

## 7.移动文件或修改文件名
mv 文件 文件夹 *移动源文件到目标文件夹中*


mv 文件A 文件B *修改文件A的名字为文件B*

# 文件查找命令
## 1.文件名查找
*语法*


find 搜索路径 -name “文件名关键词” 


*例子*


find / -name “passwd”

## 2.文件内容查找
*语法*


grep -参数 要查找的目录范围


*参数*


-n 显示查找结果所在行号


-R 递归查找目录下的所有文件


*例子*


grep zcy /etc/passwd
