# Introduction

This is the first entry of my engineering journal.

I created this website as a long-term technical portfolio to document my learning journey in robotics, control systems, and intelligent engineering.

---

# Why I Built This Website

During my transition from electrical engineering toward robotics and embodied intelligence, I realized that engineering growth requires continuous documentation and reflection.

This platform is designed to record:

- Technical experiments
- Robotics projects
- Paper reading notes
- Engineering lessons

---

# Technical Implementation

The website is built with:

- Hugo Static Site Generator
- PaperMod Theme
- GitHub Repository
- GitHub Actions CI/CD
- GitHub Pages Deployment

The workflow is:
Markdown
↓
Git Commit
↓
GitHub Actions
↓
Hugo Build
↓
GitHub Pages
